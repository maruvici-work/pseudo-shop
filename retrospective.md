## 2. What Git strategies or commands helped you most during the project?

## 3. Describe a merge conflict you encountered. What caused it and how did you resolve it?
- We both modified the main.pseudo file to create mock function calls to the functions we made in the pseudo files. This caused a merge conflict because one of us removed the initial comments and code present in the initial commit after forking the repo. Some of our own functions also conflicted. We mainly fixed it by combining lines from all branches and removing lines that are unneeded.

## 5. What did you learn about using Git in a collaborative setting?
